# Dutch-Turkish Flash Cards

A Vue 3 + TypeScript flash card application for learning Dutch-Turkish vocabulary.

## Features

- Import decks from JSON files
- Load built-in example deck
- Flash card study mode with flip card UI
- Rating buttons for spaced repetition learning
- Progress tracking and session summary
- LocalStorage persistence for decks and sessions

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Notes

- The app uses Pinia for state management.
- Tailwind CSS is configured for styling.
- Example deck is available under `public/decks/example-nl-tr.json`.
